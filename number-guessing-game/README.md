# Number Guessing Game v1.0

## Overview
Number Guessing Game is a simple **Python CLI game** where the computer randomly chooses a number and the player must guess it.

After every guess, the program tells the player whether the guess is **too high** or **too low**, helping them get closer to the correct answer. The game continues until the player guesses the number correctly.

This project is part of my **Python CLI mini-projects series** where I build small programs to practice programming concepts and problem solving. 💻🎯

---

## Features
- Random number generation between **1 and 100**
- Unlimited guessing attempts
- Helpful hints (**Too High / Too Low**)
- Tracks **number of guesses**
- Input validation to prevent crashes
- Clean and simple **command-line interface**

---

## How to Use

### 1️⃣ Clone the repository
```bash
git clone https://github.com/Arnav-Nagabhushan/number-guessing-game.git
```

### 2️⃣ Navigate to the project folder
```bash 
cd number-guessing-game
```

### 3️⃣ Run the program
```bash 
python number_guessing_game.py
```

### 4️⃣ Start guessing!

- The program will choose a number between 1 and 100.
- Keep guessing until you find the correct number.
