# Password Generator v1.0

## Overview
Password Generator is a simple **Python CLI tool** that creates strong random passwords based on a user-defined length.

The program combines **lowercase letters, uppercase letters, numbers, and symbols** to generate secure passwords. The user only needs to enter the desired password length, and the program will generate a random password instantly.

This project is part of my **Python CLI mini-projects series** where I build small programs to practice programming concepts and improve my problem-solving skills. 💻

---

## Features
- Generates **secure random passwords**
- User can choose the **desired password length**
- Includes:
  - lowercase letters
  - uppercase letters
  - numbers
  - symbols
- **Input validation** to prevent invalid entries
- Clean and simple **command-line interface**
- Structured using **functions and loops**

## How to Use

### 1️⃣ Clone the repository
```bash
git clone https://github.com/Arnav-Nagabhushan/password-generator.git
```

### 2️⃣ Navigate to the project folder
```bash
cd password-generator
```

### 3️⃣ Run the program
```bash
python password_generator_v1.0.py
```

### 4️⃣ Generate a password
- Enter the desired password length.
- The program will generate a secure random password.
