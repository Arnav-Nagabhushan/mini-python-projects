# Task Manager (To-Do List CLI) v1.0

## Overview
Task Manager is a simple **Python command-line program** that helps users manage their daily tasks. You can **view tasks**, **add new tasks**, and **remove completed tasks** all from the terminal.

This project is perfect for beginners learning Python functions, loops, and list operations.

## Features
- View all tasks with numbered listing  
- Add a new task to your task list  
- Remove a task by its number  
- Menu-driven CLI for easy navigation  
- Input validation to prevent errors  

## How to Use
1. Clone or download the repository.  
2. Run the program in a terminal using:

```bash
python task_manager_v1.py
```

3. Follow the menu options:

1 – View Tasks
2 – Add a Task
3 – Remove a Task
4 – Exit

## Learning Focus

- Using lists to store data
- Writing functions for clean code structure
- Using loops and enumerate() to display tasks
- Handling user input and errors
