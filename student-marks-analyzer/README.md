# Student Marks Analyzer v1.0

## Overview
Student Marks Analyzer is a simple **Python program** that helps students (or teachers) calculate and analyze marks for different subjects. It computes the **average**, **percentage**, identifies the **highest** and **lowest scoring subjects**, and displays a **neatly formatted report card** right in your terminal.

Perfect for beginners looking to practice **Python**, **functions**, and **basic data processing**. 💻

---

## Features
- Calculates **average marks** across subjects  
- Computes **percentage** based on total marks  
- Finds the **highest scoring subject**  
- Finds the **lowest scoring subject**  
- Displays a **clean, formatted report card**  
- Modular and easy to **extend for more subjects**

---

## How to Use

### 1️⃣ Clone the repository
```bash
git clone https://github.com/Arnav-Nagabhushan/student-marks-analyzer.git
```
### 2️⃣ Navigate to the project folder
```bash
cd student-marks-analyzer
```
### 3️⃣ Run the program
```bash
python student_marks.py
```
### 4️⃣ Follow the prompts
Enter your name and marks for each subject when prompted.

Then view your generated report card in the terminal 🎉
